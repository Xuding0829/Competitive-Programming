# 选Win、Mac还是Linux

上文说到选购的问题。因为大多数本科生应该只有一台电脑，所以选择哪一个是很需要考虑的问题。下面笔者凭借自己的经验，对每一个属性做一定的解释。仅仅是一家之言，不太表任何绝对性的观点。
